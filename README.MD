## How to use this ...

* Edit slides.md
* run ./build.sh
* open index.html

There are many [themes to choose from](https://revealjs.com/themes/). Edit `build.sh` to change your theme.

> This project requires [pandoc](https://pandoc.org).

    $ brew install pandoc
